# Vue-cms

**【请移步分支： master】**

`screenfull` 插件使用注意事项：在 `package.json` 里 `dependencies` 中添加 `"screenfull": "^3.3.2",`，然后执行 `yarn` 命令。 不要直接 `yarn add screenfull`。
